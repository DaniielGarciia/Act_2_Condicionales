Nivel 1: Validación y Condicionales (Switch / If)

Ejercicio 1

Pide la edad.
Si es menor de 18, alert de "Acceso denegado"; si es 18 o más, "Bienvenido".
Valida que no ingresen texto.

Ejercicio 2

El helado base cuesta $5.
Pregunta el topping: "Oreo" (+$2), "KitKat" (+$3) o "Brownie" (+$4).
Usa switch y muestra el precio final.

Ejercicio 3

Pide un número y usa el operador módulo % 2 con un if para decir si es par o impar.

Ejercicio 4

Pide un número del 1 al 7.
con switch, muestra el nombre del día (1 = Lunes, etc.).
Si es otro número, aviso de error.

Ejercicio 5

Pide dos números y luego una operación ("suma", "resta", "mult", "div").
Usa switch para realizar la operación elegida.

Ejercicio 6

Pide 3 notas, saca el promedio con parseFloat.
Si el promedio es >= 6, "Aprobado", si no, "Reprobado".

Ejercicio 7

Pide un valor en dólares y que elijan a qué moneda convertir (Euros, Pesos, Soles). Usa switch con tasas de cambio fijas.

Nivel 2: Bucles (El mismo ejercicio de 3 formas)

Para estos ejercicios, pídeles que los resuelvan primero con for, luego con while y luego con do-while.

Ejercicio 8

Mostrar en el HTML los números del 10 al 1.

Ejercicio 9

Pide un número $N$ y suma todos los números desde 1 hasta $N$ (Ej: si eligen 5, sumar 1+2+3+4+5).

Ejercicio 10

Mostrar todos los múltiplos de 5 que hay entre 1 y 50.

Ejercicio 11

Crea un bucle while que pida una contraseña por prompt repetidamente hasta que el usuario escriba "SESAMO123".

Ejercicio 12

Pide un número y muestra sus primeras 5 potencias (Número^1, Número^2, etc.)

Ejercicio 13

Usa un do-while para pedir números al usuario. El bucle se detiene cuando el usuario ingresa un número negativo.

Nivel 3: Estructuras de Datos y Lógica Combinada

Ejercicio 14

Crea un array vacío.
Usa un bucle for para pedir 5 productos por prompt y agregarlos al array con .push().
Al final, muéstralos en el HTML.

Ejercicio 15

Ten un array con 5 nombres.
Pide un nombre al usuario y usa un bucle para verificar si el nombre existe en la lista.

Ejercicio 16

Crea un objeto auto con propiedades (marca, modelo, año).
Pide al usuario que actualice el año mediante un prompt y muestra el objeto actualizado.

Ejercicio 17

Crea un array.
Pide los datos de 3 estudiantes (nombre y nota).
Guarda cada uno como objeto dentro del array.

Ejercicio 18

Pide precios de productos hasta que el usuario escriba "0".
Si el total supera los $100, aplica un 10% de descuento usando un if.

Ejercicio 19

Usa un do-while para mostrar un menú: "1. Saludar, 2. Despedirse, 3. Salir".
El bucle solo termina cuando elijan la opción 3.

Ejercicio 20

Crea un array de 5 números.
Pide al usuario un número y una posición (0 a 4).
Reemplaza el valor en esa posición y muestra el array final.
